## Why Komputer.AI

<CardGrid stagger>
  <Card title="Kubernetes-native" icon="seti:kubernetes">
    Agents are first-class CRDs. The operator reconciles them like any other resource — Pods, PVCs, lifecycle, and all.
  </Card>
  <Card title="Stateless API" icon="seti:json">
    No database. CR `.status` fields are the source of truth; Redis is just a transport. Restart anything, lose nothing.
  </Card>
  <Card title="Real-time streaming" icon="bars">
    REST for control, WebSocket for live event streams from every agent — token-by-token output, tool calls, costs.
  </Card>
  <Card title="Squads & sub-agents" icon="puzzle">
    Multi-agent shared workspaces via co-located Pods. Manager agents create and orchestrate workers via MCP tools.
  </Card>
  <Card title="Built-in observability" icon="approve-check">
    Structured logs (JSON or human), Prometheus metrics, ServiceMonitors, and a Grafana dashboard ship with the chart.
  </Card>
  <Card title="MCP connectors" icon="setting">
    Plug in MCP servers per agent or per template. Credentials managed via secrets, lifecycle managed by the operator.
  </Card>
</CardGrid>
